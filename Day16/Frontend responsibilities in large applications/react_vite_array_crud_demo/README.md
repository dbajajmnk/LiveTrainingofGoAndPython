# React + Vite Array CRUD Demo

Beginner-friendly project to demonstrate CRUD on an array using React state.

## Concepts covered
- Create item
- Read item list
- Update item
- Delete item
- Controlled form
- Lifting state into a custom hook
- Separate files in a small Vite project

## Run
```bash
npm install
npm run dev
```

## Folder Structure
```text
src/
  components/
    ItemForm.jsx
    ItemList.jsx
  hooks/
    useItems.js
  App.jsx
  main.jsx
  styles.css
```
