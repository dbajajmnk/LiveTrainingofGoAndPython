# Vite React Structured Frontend Demo

Beginner-friendly project showing frontend role in a large-scale system.

## Included concepts
- Separate files
- Service layer
- Custom hook
- Presentational component
- Page component
- Shared components
- Vite + React setup

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Folder structure
```text
src/
  components/
    OrderList.jsx
  features/
    orders/
      OrderPage.jsx
      orderService.js
      useOrders.js
  shared/
    ErrorMessage.jsx
    Loader.jsx
  App.jsx
  main.jsx
  styles.css
```
