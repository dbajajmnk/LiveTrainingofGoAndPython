## With Global Environment
-   Install required dependencies
    -   pip install [fastapi](https://pypi.org/project/fastapi/)
    -   pip install [uvicorn](https://pypi.org/project/uvicorn/)
    -   pip install [pydantic](https://pypi.org/project/pydantic/)

## Create a file in your project with name 
app.py write your code there

## Run your first project
py -m uvicorn app:app --reload
